# Change Font of .welcome Class

## Overview
Update the font styling of the .welcome CSS class in the main stylesheet to a different font family.

## Goals
- Modify the font property of the .welcome class to a new font family
- Ensure the change is applied consistently across all elements using the .welcome class

## Requirements

### 1. Font Styling Changes

#### 1.1 Update .welcome class font
**Priority:** must-have

**Acceptance Criteria:**
- [ ] The .welcome class has a new font family applied
- [ ] The font change is visible on all elements with the .welcome class
- [ ] The change is saved in the main stylesheet
